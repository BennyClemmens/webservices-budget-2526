## Start the project

### Development

- create a .env file with this content:

```ini
NODE_ENV=development
PORT=3000
CORS_ORIGIN=["http://localhost:5173"]
CORS_MAX_AGE=10800
```

You can configure another PORT if this one is not convenient on your system.

- `pnpm install`
- `pnpm start:dev`
